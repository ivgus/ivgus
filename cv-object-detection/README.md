# ML-модель детекции автомобилей на изображениях
Проект представляет собой систему компьютерного зрения на основе свёрточных нейронных сетей для классификации и детектирования объектов на изображениях. Собственное архитектурное решение.
Модули: бинарная классификация объектов (человек/транспорт), детектирование для определения координат объектов на изображении, обученные модели на базе свёрточных нейронных сетей.

Точность классификации: более 80% (accuracy). Ошибка детектирования: средняя абсолютная ошибка (MAE) менее 5%.

Обучение на базе открытого датасета изображений с видеорегистратора.

Технологии: Python, Numpy, Keras, TensorFlow, Pickle, Matplotlib
