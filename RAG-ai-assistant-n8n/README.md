# RAG-система с AI-агентом для консультации
LLM агент с основным workflow и вспомогательным workflow (построение эмбеддингов) на n8n на базе моделей gpt-oss-120b (OpenAI, free) для ai-ассистента и трансформера paraphrase-multilingual-MiniLM-L12-v2 (Hugging Face) для построения эмбеддингов и их ранжирования.
БД Qdrant с открытой документацией Росстата по торговле (настраиваемая векторизация в процессе работы моделью с Hugging Face). Для базы данных создан образ в Docker.
Интерактивный чат в среде n8n.

Технологии: n8n, Node.js, Qdrant, Docker, json, REST API, LLM модели

Запуск: Импортируйте json workflows в n8n. В блоке OpenRouter Chat Model вставьте свой Api ключ от аккаунта в OpenRouter (агрегатор LLM). В блоке Qdrant Vector Store вставьте свой Api ключ связи с БД Qdrant. В Qdrant добавьте данные с официального сайта Росстата в формате pdf, вставив их в папку, указанную в вспомогательном workflow в ноде Read File from Disk. Они векторизуются и добавятся в БД в ходе работы этого workflow по нажатию на триггер "Execute workflow". Для начала работы подготовьте среду по указаниям выше и напишите сообщение в чат (например "Напиши, о чём ты мне можешь рассказать.").

Настраивается степень креативности агента, прописаны системные промты и установлено хранилище контекста прошлых сообщений и ответов.
