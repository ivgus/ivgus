# Реляционная база данных в среде Access с SQL запросами
База данных и приложение для анализа и сравнения зарубежного и отечественного программного обеспечения в контексте импортозамещения.
Система позволяет вести учет ПО, вендоров и их характеристик, а также проводить сравнительный анализ возможностей замены иностранного ПО на российское.
UI (интерфейс) написан на Visual Basic for Applications (VBA).

Основные компоненты:
- База данных на платформе MS Access
- Пользовательский интерфейс с формами для работы с данными
- Система запросов для анализа информации
- Модуль авторизации для защиты данных

Функциональные возможности:
- Учет и управление информацией о зарубежных и отечественных программных продуктах, компаниях-разработчиках (вендорах) функционале ПО, характеристиках и стоимости продуктов
- Аналитика (сравнение цен на ПО, анализ соответствия функционала, оценка возможностей импортозамещения, формирование отчетов)

Технологии: MS Access, SQL
