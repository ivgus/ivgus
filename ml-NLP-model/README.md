# Text Analysis ML Assistant
Программа для интеллектуальной обработки текстовых запросов на русском языке с использованием нейронных сетей.
Система анализирует неформализованные пользовательские запросы и автоматически определяет, какую информацию и в какое поле базы данных необходимо внести.
Классификация запросов по типу данных (имена, телефонные номера, email-адреса), извлечение сущностей из текста запроса, нормализация данных (приведение имен к именительному падежу).
Система состоит из двух основных компонентов:
нейронная сеть для классификации типа данных, нейронная сеть для извлечения целевых сущностей. Для обработки никогда не виденных слов словарь моделей дообучается.

Нужный датасет был сгенерирован языковой моделью Copilot по опорным примерам.

Ошибки последовательных классификаций: Categorical Crossentropy 0.25, Binary Crossentropy 0.001.
Точность моделей (accuracy): 0.92 и 1.0.

Технологии: Python, NLTK, Numpy, Gensim, Keras, TensorFlow, Itertools, Matplotlib
